# Declastruct Squarespace Domain - Distilled Design

## 1. Use Cases

Based on the wish in `0.wish.md`, the following use cases are required:

### UC1: Enumerate All Domains
**Goal:** See all 300+ domains currently in the Squarespace account.
**Actor:** Automation script
**Flow:**
1. Authenticate with Squarespace (via Playwright)
2. Navigate to domains dashboard
3. Scrape all domain entries
4. Return list of `DeclaredSquarespaceDomain` objects

### UC2: View Domain DNS Records
**Goal:** Export/document current DNS configuration before transfer.
**Actor:** Automation script
**Flow:**
1. Navigate to specific domain's DNS settings
2. Scrape preset records and custom records
3. Return list of `DeclaredSquarespaceDnsRecord` objects

### UC3: Prepare Domain for Transfer
**Goal:** Unlock domain and disable DNSSEC to enable transfer-out.
**Actor:** Automation script
**Flow:**
1. Check if domain has DNSSEC enabled → disable if so
2. Check if domain is locked → unlock if so
3. Verify domain is transferable (no 60-day locks)

### UC4: Request Transfer Authorization Code
**Goal:** Initiate transfer-out by requesting EPP/auth code.
**Actor:** Automation script
**Flow:**
1. Navigate to domain detail page
2. Click "Request transfer code" button
3. Re-authenticate when prompted
4. Wait for confirmation
5. Email arrives within 24 hours (out-of-band)

### UC5: Batch Transfer Domains
**Goal:** Automate transfer-out for 300+ domains.
**Actor:** Automation orchestrator
**Flow:**
1. Get all domains (UC1)
2. For each domain:
   - Export DNS records (UC2)
   - Prepare for transfer (UC3)
   - Request transfer code (UC4)
3. Track transfer status

### Usage Example

```typescript
import { declastruct } from 'declastruct';
import { getDeclastructSquarespaceProvider } from 'declastruct-squarespace';

// Initialize provider with Playwright browser context
const provider = await getDeclastructSquarespaceProvider({
  context: {
    squarespace: {
      browser: playwrightBrowserContext,
      credentials: {
        email: process.env.SQUARESPACE_EMAIL,
        password: process.env.SQUARESPACE_PASSWORD,
      },
    },
  },
});

// Use Case 1: Get all domains
const domains = await provider.DeclaredSquarespaceDomain.get.all({});

// Use Case 2: Get DNS for a domain
const dnsRecords = await provider.DeclaredSquarespaceDnsRecord.get.all({
  by: { domain: { name: 'example.com' } },
});

// Use Case 3: Prepare domain for transfer (upsert)
await provider.DeclaredSquarespaceDomain.set.upsert({
  name: 'example.com',
  isLocked: false,
  dnssecEnabled: false,
});

// Use Case 4: Request transfer code
await provider.DeclaredSquarespaceDomainTransferRequest.set.findsert({
  domain: { name: 'example.com' },
});
```

## 3. Domain Objects

### 3.1 DeclaredSquarespaceDomain

```typescript
/**
 * @what - A domain registered with or managed by Squarespace
 * @why - Represents the primary entity for domain management and transfer operations
 * @identity - Uniquely identified by domain name
 * @ref - Can be referenced by name
 */
export interface DeclaredSquarespaceDomain {
  /**
   * .what - The domain name (e.g., "example.com")
   * .why - Primary identifier for the domain
   */
  name: string;

  /**
   * .what - Domain status
   * .why - Indicates if domain is active, expired, or pending
   */
  status: DeclaredSquarespaceDomainStatus;

  /**
   * .what - Whether domain transfer lock is enabled
   * .why - Must be false (unlocked) to transfer domain out
   */
  isLocked: boolean;

  /**
   * .what - Reason domain cannot be unlocked (if applicable)
   * .why - Indicates 60-day registration/transfer lock or contact change lock
   */
  lockReason?: DeclaredSquarespaceDomainLockReason | null;

  /**
   * .what - The registrar of record
   * .why - Different registrars have different transfer flows
   */
  registrar: DeclaredSquarespaceRegistrarType;

  /**
   * .what - When the domain registration expires
   * .why - Expired domains have 30-day transfer window
   */
  expirationDate: string; // ISO date string

  /**
   * .what - Whether auto-renewal is enabled
   * .why - Informational
   */
  autoRenew: boolean;

  /**
   * .what - Whether DNSSEC is enabled
   * .why - Must be disabled before transfer
   */
  dnssecEnabled: boolean;

  /**
   * .what - When the domain was registered/transferred to Squarespace
   * .why - Used to determine if 60-day lock applies
   */
  createdAt?: string; // ISO date string, readonly
}

export class DeclaredSquarespaceDomain extends DomainEntity<DeclaredSquarespaceDomain> {
  public static primary = [] as const; // No natural primary key from Squarespace
  public static unique = ['name'] as const;
  public static metadata = [] as const;
  public static readonly = ['createdAt', 'registrar', 'status', 'lockReason'] as const;
}
```

### 3.2 DeclaredSquarespaceDnsRecord

```typescript
/**
 * @what - A DNS record associated with a Squarespace domain
 * @why - Required for exporting DNS configuration before transfer
 * @identity - Uniquely identified by domain + type + host
 * @ref - Can be referenced by domain + type + host combination
 */
export interface DeclaredSquarespaceDnsRecord {
  /**
   * .what - Reference to the parent domain
   */
  domain: RefByUnique<typeof DeclaredSquarespaceDomain>;

  /**
   * .what - DNS record type (A, AAAA, CNAME, MX, TXT, etc.)
   */
  type: DeclaredSquarespaceDnsRecordType;

  /**
   * .what - The host/subdomain (e.g., "@" for root, "www", "mail")
   */
  host: string;

  /**
   * .what - The record value (IP, hostname, text content, etc.)
   */
  value: string;

  /**
   * .what - Time to Live in seconds
   * .why - Controls DNS caching behavior
   */
  ttl: number;

  /**
   * .what - Priority value (for MX, SRV records)
   */
  priority?: number | null;

  /**
   * .what - Whether this is a Squarespace preset or custom record
   * .why - Presets may have special handling for deletion
   */
  isPreset: boolean;
}

export class DeclaredSquarespaceDnsRecord extends DomainEntity<DeclaredSquarespaceDnsRecord> {
  public static primary = [] as const;
  public static unique = ['domain', 'type', 'host'] as const;
  public static metadata = [] as const;
  public static readonly = ['isPreset'] as const;
}
```

## 6. Context Types

### 6.1 ContextPlaywrightBrowser

```typescript
import type { Browser, BrowserContext, Page } from 'playwright';

/**
 * @what - Context carrying Playwright browser, context, and reusable pages
 * @why - Maximizes reuse of browser resources across operations, avoiding
 *        repeated authentication and navigation overhead
 */
export interface ContextPlaywrightBrowser {
  playwright: {
    /**
     * The Playwright browser instance
     */
    browser: Browser;

    /**
     * The browser context with authenticated session cookies
     * @note - Should be created once and reused across all operations
     */
    context: BrowserContext;

    /**
     * Cache of open pages keyed by purpose/URL pattern
     * @why - Avoids reopening pages for sequential operations on same view
     */
    pages: {
      /**
       * The domains dashboard list page
       * @url - account.squarespace.com/domains
       */
      domainsList?: Page;

      /**
       * Currently open domain detail page
       * @note - Keyed by domain name to detect if navigation needed
       */
      domainDetail?: {
        domainName: string;
        page: Page;
      };

      /**
       * Currently open DNS settings page
       * @note - Keyed by domain name to detect if navigation needed
       */
      dnsSettings?: {
        domainName: string;
        page: Page;
      };
    };

    /**
     * Configuration for page management
     */
    config: {
      /**
       * Whether to reuse pages or create fresh ones for each operation
       * @default true
       */
      reusePages: boolean;

      /**
       * Timeout for page operations in milliseconds
       * @default 30000
       */
      timeout: number;

      /**
       * Delay between operations to avoid rate limiting (ms)
       * @default 1000
       */
      operationDelay: number;
    };
  };
}
```

### 6.4 Context Factory

```typescript
/**
 * Create a ContextSquarespaceApi with sensible defaults
 */
export const createSquarespaceContext = async (input: {
  credentials: {
    email: string;
    password: string;
    totpSecret?: string;
  };
  browserOptions?: {
    headless?: boolean;  // Default: true
    slowMo?: number;     // Default: 0
  };
}): Promise<ContextSquarespaceApi> => {
  const browser = await chromium.launch({
    headless: input.browserOptions?.headless ?? true,
    slowMo: input.browserOptions?.slowMo ?? 0,
  });

  const context = await browser.newContext({
    userAgent: 'Mozilla/5.0 ...', // Realistic user agent
    viewport: { width: 1920, height: 1080 },
  });

  const baseUrl = 'https://account.squarespace.com';

  return {
    playwright: {
      browser,
      context,
      pages: {},
      config: {
        reusePages: true,
        timeout: 30000,
        operationDelay: 1000,
      },
    },
    squarespace: {
      credentials: input.credentials,
      baseUrl,
      urls: {
        domainsList: `${baseUrl}/domains`,
        domainDetail: (name) => `${baseUrl}/domains/${name}`,
        dnsSettings: (name) => `${baseUrl}/domains/${name}/dns`,
      },
    },
  };
};

/**
 * Cleanup context resources
 */
export const destroySquarespaceContext = async (
  context: ContextSquarespaceApi,
): Promise<void> => {
  // Close all cached pages
  for (const page of Object.values(context.playwright.pages)) {
    if (page && 'close' in page) {
      await (page as Page).close();
    } else if (page && 'page' in page) {
      await page.page.close();
    }
  }

  // Close browser context and browser
  await context.playwright.context.close();
  await context.playwright.browser.close();
};
```

### 6.5 Page Reuse Helpers

```typescript
/**
 * Get or create the domains list page
 */
export const getDomainsListPage = async (
  context: ContextSquarespaceApi,
): Promise<Page> => {
  // Return cached page if available and valid
  if (context.playwright.pages.domainsList) {
    const page = context.playwright.pages.domainsList;
    if (!page.isClosed()) {
      // Ensure we're on the right URL
      if (!page.url().includes('/domains')) {
        await page.goto(context.squarespace.urls.domainsList);
      }
      return page;
    }
  }

  // Create new page
  const page = await context.playwright.context.newPage();
  await page.goto(context.squarespace.urls.domainsList);

  // Cache for reuse
  if (context.playwright.config.reusePages) {
    context.playwright.pages.domainsList = page;
  }

  return page;
};

/**
 * Get or create a domain detail page
 */
export const getDomainDetailPage = async (
  context: ContextSquarespaceApi,
  domainName: string,
): Promise<Page> => {
  const cached = context.playwright.pages.domainDetail;

  // Return cached if same domain and valid
  if (cached?.domainName === domainName && !cached.page.isClosed()) {
    return cached.page;
  }

  // Close old cached page if different domain
  if (cached && !cached.page.isClosed()) {
    await cached.page.close();
  }

  // Create new page
  const page = await context.playwright.context.newPage();
  await page.goto(context.squarespace.urls.domainDetail(domainName));

  // Cache for reuse
  if (context.playwright.config.reusePages) {
    context.playwright.pages.domainDetail = { domainName, page };
  }

  return page;
};
```

## 8. Implementation Notes

### Playwright Automation Considerations

1. **Session Management**: Maintain authenticated Playwright browser context across operations.

2. **Reauthentication**: Squarespace prompts for password/2FA for sensitive operations (DNS changes, transfer codes). Need to detect and handle these popups.

3. **Pagination**: Domain list may span multiple pages for 300+ domains. Need infinite scroll or pagination handling.

4. **Rate Limiting**: Unknown if Squarespace rate-limits UI interactions. May need to add delays between operations.

5. **Error States**: Handle UI error messages (e.g., "Domain is locked", "60-day restriction").

6. **Idempotency**: Since we're scraping UI, operations should check current state before acting to maintain idempotency.

### Transfer Workflow

The recommended batch transfer workflow:

```typescript
async function batchTransferDomains(provider: DeclastructSquarespaceProvider) {
  // 1. Get all domains
  const domains = await provider.DeclaredSquarespaceDomain.get.all({});

  // 2. For each domain
  for (const domain of domains) {
    // 2a. Export DNS records (save externally)
    const dnsRecords = await provider.DeclaredSquarespaceDnsRecord.get.all({
      by: { domain: { name: domain.name } },
    });
    await saveDnsRecords(domain.name, dnsRecords);

    // 2b. Check if transferable
    if (domain.lockReason) {
      console.log(`Skipping ${domain.name}: ${domain.lockReason}`);
      continue;
    }

    // 2c. Prepare for transfer
    await provider.DeclaredSquarespaceDomain.set.upsert({
      name: domain.name,
      isLocked: false,
      dnssecEnabled: false,
    });

    // 2d. Request transfer code
    await provider.DeclaredSquarespaceDomainTransferRequest.set.findsert({
      domain: { name: domain.name },
    });

    // 2e. Wait between requests to avoid rate limiting
    await sleep(5000);
  }
}
```
