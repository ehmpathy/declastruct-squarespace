## issues found and fixed

### violation 1: else branch in scrollToLoadAllDomains.ts

**rule:** `rule.forbid.else-branches`

**issue:** `scrollToLoadAllDomains.ts` line 34 used `else` branch.

**before:**
```ts
if (currentCount > previousCount) {
  previousCount = currentCount;
  noNewDomainsAttempts = 0;
} else {
  noNewDomainsAttempts++;
}
```

**after:**
```ts
const hasNewDomains = currentCount > previousCount;
if (hasNewDomains) {
  previousCount = currentCount;
  noNewDomainsAttempts = 0;
}
if (!hasNewDomains) {
  noNewDomainsAttempts++;
}
```

**status:** fixed (prior session).

### violation 2: else branch in waitForHumanSignal.ts

**rule:** `rule.forbid.else-branches`

**issue:** `waitForHumanSignal.ts` line 52 used `else` branch in poll loop.

**before:**
```ts
const check = () => {
  if (fs.existsSync(filePath)) {
    fs.unlinkSync(filePath);
    fulfill();
  } else {

  }

```

**after:**
```ts
const check = () => {
  const fileExists = fs.existsSync(filePath);

  setTimeout(check, 500);

```
