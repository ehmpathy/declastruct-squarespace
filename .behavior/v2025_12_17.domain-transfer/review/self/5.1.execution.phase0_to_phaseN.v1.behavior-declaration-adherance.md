## deviations found and fixed

### deviation 1: awaited verification check was absent (usecase.5)

**criteria says:**
> given('a domain unlocked but with awaited verification')
>   when('user requests transfer code')
>     then('returns error: domain has awaited verification')

**issue:** setTransferRequest.ts checked for locked domains but did not check for PENDING status (awaited verification).

**file:** `src/domain.operations/domainTransferRequest/setTransferRequest.ts`

**fix:** added status check after lock check
```ts
// check for awaited verification (per blackbox criteria usecase.5)
if (domainFound.status === 'PENDING') {
  throw new BadRequestError('domain has awaited verification', {
    domain: domainName,
    status: domainFound.status,
    hint: 'complete domain verification before transfer request',
  });
}
```

**status:** fixed

### deviation 2: review incorrectly referenced bottleneck location

**issue:** review claimed bottleneck was declared in withNewLoggedInBrowserPage.ts but it's actually in getSquarespaceAgentOptions.ts.

**fix:** updated review to reference correct file location (getSquarespaceAgentOptions.ts:64-67).

**status:** fixed (documentation only)

### deviation 3: session refresh errors lacked context (usecase.7)

**criteria says:**
> given('session refresh fails mid-batch')
>   when('next operation is attempted')
>     then('returns error: session refresh failed with reason')

**issue:** withNewLoggedInBrowserPage.ts propagated raw auth errors without context about session refresh failure.

**file:** `src/access/sdks/squarespace.via.playwright/wrappers/withNewLoggedInBrowserPage.ts`

**fix:** added try/catch to wrap auth errors with session refresh context
```ts
// wrap auth errors to clarify this is a session refresh failure (per criteria usecase.7)
if (error instanceof BadRequestError) {
  throw new BadRequestError('session refresh failed', {
    reason: sessionStatus.reason,
    authError: error.message,
    hint: 'fix credentials and resume batch',
  });
}
```

**status:** fixed
