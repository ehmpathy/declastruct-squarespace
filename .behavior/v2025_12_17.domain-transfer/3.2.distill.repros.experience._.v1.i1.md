# enframe: playloops for declastruct-squarespace

## objective

frame the playloops that enable rapid feedback for domain transfer. each playloop reproduces a user journey.

---

## experience reproduction matrix

| experience | entry point | user actions | expected outcome | how to assert |
|------------|-------------|--------------|------------------|---------------|
| enumerate domains | `provider.daos.DomainRegistration.get.all()` | call api | list of all domains returned | array length > 0, each has name |
| check domain detail | `provider.daos.DomainRegistration.get.one.byUnique()` | lookup by name | domain with lock status, registrar | entity shape matches |
| check dns records | `provider.daos.DnsRecord.get.all({ domain })` | query by domain ref | list of dns records | array of records with type, value |
| unlock domain | `provider.daos.DomainRegistration.set.upsert({ isLocked: false })` | mutation | domain unlocked | isLocked === false after |
| disable dnssec | `provider.daos.DomainRegistration.set.upsert({ dnssecEnabled: false })` | mutation | dnssec disabled | dnssecEnabled === false after |
| request transfer code | `provider.daos.TransferRequest.set.findsert()` | mutation | transfer requested | request entity returned |
| idempotent re-request | `provider.daos.TransferRequest.set.findsert()` again | mutation | same request returned | no duplicate, same requestedAt |
| batch plan | `declastruct plan --wish resources.ts` | cli command | plan json file created | file exists, has changes array |
| batch apply | `declastruct apply --plan plan.json` | cli command | changes applied | re-plan shows all KEEP |

## playloop code sketches

### playloop 1: enumerate domains

**what it tests:** api returns list of domains with expected shape

```ts
describe('enumerate domains', () => {
  given('[case1] squarespace account with domains', () => {
    const scene = useBeforeAll(async () => {
      const context = getSampleSquarespaceContext();
      const domains = await provider.daos.DeclaredSquarespaceDomainRegistration.get.all({}, context);
      return { domains };
    });

    when('[t0] domains are retrieved', () => {
      then('returns non-empty array', () => {
        expect(scene.domains.length).toBeGreaterThan(0);
      });

      then('each domain has a name', () => {
        scene.domains.forEach((d) => {
          expect(typeof d.name).toBe('string');
          expect(d.name).toMatch(/\./);  // looks like domain
        });
      });

      then('each domain has lock status', () => {
        scene.domains.forEach((d) => {
          expect(typeof d.isLocked).toBe('boolean');
        });
      });
    });
  });
});
```

**tools required:** jest, test-fns, real squarespace account

### playloop 2: unlock domain

**what it tests:** mutation changes domain state

```ts
describe('unlock domain', () => {
  given('[case1] a locked domain', () => {
    const scene = useBeforeAll(async () => {
      const context = getSampleSquarespaceContext();
      const domains = await provider.daos.DeclaredSquarespaceDomainRegistration.get.all({}, context);
      const lockedDomain = domains.find((d) => d.isLocked && !d.lockReason);
      if (!lockedDomain) throw new Error('no unlockable domain found');
      return { context, domain: lockedDomain };
    });

    when('[t1] set isLocked to false', () => {
      const result = useThen('domain is unlocked', async () => {
        return provider.daos.DeclaredSquarespaceDomainRegistration.set.upsert(
          { entity: new DeclaredSquarespaceDomainRegistration({ ...scene.domain, isLocked: false }) },
          scene.context
        );
      });

      then('returned domain shows unlocked', () => {
        expect(result.isLocked).toBe(false);
      });
    });

    afterAll(async () => {
      // cleanup: re-lock domain
      await provider.daos.DeclaredSquarespaceDomainRegistration.set.upsert(
        { entity: new DeclaredSquarespaceDomainRegistration({ ...scene.domain, isLocked: true }) },
        scene.context
      );
    });
  });
});
```

**tools required:** jest, test-fns, test domain that can be locked/unlocked

### playloop 3: idempotent transfer request

**what it tests:** findsert returns same request on retry

```ts
describe('transfer request idempotency', () => {
  given('[case1] unlocked domain ready for transfer', () => {
    const scene = useBeforeAll(async () => {
      const context = getSampleSquarespaceContext();
      // find or setup test domain
      return { context, domain };
    });

    when('[t1] request transfer code', () => {
      const requestFirst = useThen('first request succeeds', async () => {
        return provider.daos.DeclaredSquarespaceDomainTransferRequest.set.findsert(
          { entity: new DeclaredSquarespaceDomainTransferRequest({ domain: refByUnique(scene.domain) }) },
          scene.context
        );
      });

      then('returns request entity', () => {
        expect(requestFirst.status).toBeDefined();
        expect(requestFirst.requestedAt).toBeDefined();
      });
    });

    when('[t2] request transfer code again', () => {
      const requestSecond = useThen('second request succeeds', async () => {
        return provider.daos.DeclaredSquarespaceDomainTransferRequest.set.findsert(
          { entity: new DeclaredSquarespaceDomainTransferRequest({ domain: refByUnique(scene.domain) }) },
          scene.context
        );
      });

      then('returns same request (idempotent)', () => {
        expect(requestSecond.requestedAt).toEqual(requestFirst.requestedAt);
      });
    });
  });
});
```

**tools required:** jest, test-fns, test domain for transfer

### playloop 4: batch workflow

**what it tests:** full declastruct plan/apply workflow

```ts
describe('batch domain transfer workflow', () => {
  given('[case1] account with multiple domains', () => {
    when('[t1] declastruct plan', () => {
      const prep = useBeforeAll(async () => {
        execSync(`npx declastruct plan --wish ${resourcesFile} --into ${planFile}`, {
          stdio: 'inherit',
          env: process.env,
        });
        return { plan: JSON.parse(readFileSync(planFile, 'utf-8')) };
      });

      then('plan file created', () => {
        expect(existsSync(planFile)).toBe(true);
      });

      then('plan has changes array', () => {
        expect(prep.plan).toHaveProperty('changes');
      });
    });

    when('[t2] declastruct apply', () => {
      beforeAll(async () => {
        execSync(`npx declastruct apply --plan ${planFile}`, {
          stdio: 'inherit',
          env: process.env,
        });
      });

      then('apply completes without error', () => {
        // if we got here, apply succeeded
        expect(true).toBe(true);
      });
    });

    when('[t3] re-plan to verify idempotency', () => {
      const verify = useBeforeAll(async () => {
        execSync(`npx declastruct plan --wish ${resourcesFile} --into ${verifyPlanFile}`, {
          stdio: 'inherit',
          env: process.env,
        });
        return { plan: JSON.parse(readFileSync(verifyPlanFile, 'utf-8')) };
      });

      then('all resources show KEEP', () => {
        const nonKeepChanges = verify.plan.changes.filter((r) => r.action !== 'KEEP');
        expect(nonKeepChanges).toHaveLength(0);
      });
    });
  });
});
```

**tools required:** jest, test-fns, declastruct cli, resources file

## gaps and blockers

### gap 1: transfer completion not testable

**what is blocked:** cannot verify transfer completes end-to-end

**why blocked:** transfer takes up to 15 days, requires email confirmation, involves external registrar

**resolution:** test only the request phase; trust icann/squarespace for completion

**blocker status:** not a blocker (out of scope)

### gap 2: email delivery not testable

**what is blocked:** cannot verify auth code email arrives

**why blocked:** email delivery is squarespace's responsibility

**resolution:** verify transfer request was submitted; trust email delivery

**blocker status:** not a blocker (out of scope)

### gap 3: 60-day lock not bypassable

**what is blocked:** cannot unlock domains within 60-day lock period

**why blocked:** icann policy enforced by squarespace

**resolution:** test with older domains; validate lockReason is returned correctly

**blocker status:** not a blocker (expected behavior)

## playloop enables rapid feedback because

- **seconds to reproduce:** run single integration test in 30-60 seconds
- **friction felt early:** selector failures, auth issues, rate limits caught in test
- **defects caught before user:** cache bugs, idempotency failures visible in playloop
- **permanent spec:** playloop serves as executable documentation of expected behavior

## scope

### happy paths
- enumerate domains returns list
- lookup domain returns detail
- unlock domain changes state
- disable dnssec changes state
- request transfer code creates request
- batch plan creates plan file
- batch apply executes changes
- re-plan shows idempotent state

### guard paths
- findsert returns extant request (no duplicate)
- upsert returns current state if no change (idempotent)
- locked domain returns lockReason (explains why)
- nonexistent domain returns null (not error)
- cache returns fast on second query (optimization)
- session refreshes on timeout (recovery)
