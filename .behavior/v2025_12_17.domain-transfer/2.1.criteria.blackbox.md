# blackbox criteria: declastruct-squarespace domain transfer

## usecase.1 = enumerate domains

```
given('a squarespace account with domains')
  when('user requests all domains')
    then('returns list of all domain registrations')
      sothat('user knows what domains exist in the account')
    then('each domain includes name, lock status, dnssec status, registrar type')
      sothat('user can assess transfer readiness')
    then('result is cached for subsequent calls')
      sothat('repeated queries are fast')

given('a squarespace account with no domains')
  when('user requests all domains')
    then('returns empty list')
      sothat('user knows the account has no domains')
```

## usecase.6 = batch domain transfer workflow

```
given('300 domains in squarespace account')
  when('user declares desired state: all unlocked, dnssec disabled, transfer codes requested')
    then('plan shows required changes for each domain')
      sothat('user can review before execution')

  when('user applies the plan')
    then('domains are processed sequentially')
      sothat('rate limits and detection risks are minimized')
    then('each domain is unlocked')
    then('each domain has dnssec disabled')
    then('transfer code is requested for each domain')
    then('progress is visible')
      sothat('user knows batch is in progress')

  when('user re-runs plan after apply')
    then('plan shows no changes needed (all KEEP)')
      sothat('user can verify all operations completed')

given('batch fails mid-execution at domain 150')
  when('user re-applies the plan')
    then('completed domains are skipped (idempotent)')
    then('failed domain is retried')
    then('rest of domains are processed')
      sothat('partial failures are recoverable')
```

## usecase.8 = bot detection resilience

```

  when('operations are executed')
    then('stealth plugin masks automation signals')
      sothat('captcha trigger rate is reduced')
    then('execution pace avoids rate-based detection')
      sothat('batch operations succeed')

  when('captcha is detected')
    then('system pivots to headful browser automatically')
      sothat('checkbox can be clicked without human')
    then('robot clicks checkbox in headful mode')
      sothat('challenge is solved automatically')
    then('system pivots back to headless after solve')
      sothat('batch continues without interruption')
    then('session is preserved across pivot')
      sothat('no re-authentication required')

    then('system prompts human to solve puzzle')
      sothat('human can intervene when needed')
    then('system waits for human signal')
      sothat('automation resumes after solve')
    then('session is preserved after human solve')
      sothat('batch continues from where it left off')

  when('captcha solve fails')
    then('returns clear error: bot detection triggered')
      sothat('user can take manual action')
```
