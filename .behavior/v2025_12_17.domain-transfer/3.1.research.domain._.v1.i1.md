# research: domain

## domain objects

### entities

#### DeclaredSquarespaceDomainRegistration

the root entity — a domain registered with or managed by squarespace.

| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| name | string | no (unique key) | the domain name (e.g., "example.com") |
| status | DeclaredSquarespaceDomainRegistrationStatus | no | active, expired, or queued |
| isLocked | boolean | yes | must be false to transfer |
| lockReason | DeclaredSquarespaceDomainLockReason \| null | no | why domain cannot be unlocked |
| registrar | DeclaredSquarespaceRegistrarType | no | registrar of record |
| expirationDate | string | no | ISO date string |
| autoRenew | boolean | yes | whether auto-renewal enabled |
| dnssecEnabled | boolean | yes | must be disabled before transfer |
| createdAt | string \| null | no | ISO date string |

**identity**: unique by `name`

#### DeclaredSquarespaceDomainDnsRecord

a dns record associated with a squarespace domain.

| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| domain | RefByUnique<DomainRegistration> | no (parent ref) | reference to parent domain |
| type | DeclaredSquarespaceDomainDnsRecordType | no | A, AAAA, CNAME, MX, TXT, etc. |
| host | string | no | subdomain ("@" for root, "www", "mail") |
| value | string | yes | IP, hostname, or text content |
| ttl | number | yes | time to live in seconds |
| priority | number \| null | yes | for MX, SRV records |
| isPreset | boolean | no | squarespace preset or custom |

**identity**: unique by `[domain, type, host]`

**parent reference**: `domain` references `DeclaredSquarespaceDomainRegistration` by unique key

#### DeclaredSquarespaceDomainTransferRequest

a request to transfer a domain out of squarespace.

| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| domain | RefByUnique<DomainRegistration> | no (unique key) | reference to domain for transfer |
| requestedAt | string | no | ISO date string |
| status | DeclaredSquarespaceTransferRequestStatus | no | REQUESTED, CODE_SENT, IN_PROGRESS, etc. |

**identity**: unique by `domain`

**note**: authorization code is delivered via email, not directly retrievable

### literals

#### DeclaredSquarespaceDomainRegistrationStatus

domain registration status values.

| value | description |
|-------|-------------|
| ACTIVE | domain is registered and operational |
| EXPIRED | domain registration has expired |
| PENDING | domain registration is queued |

#### DeclaredSquarespaceRegistrarType

registrar entities in squarespace ecosystem.

| value | description |
|-------|-------------|
| SQUARESPACE_DOMAINS_LLC | native squarespace registrations |
| SQUARESPACE_DOMAINS_II_LLC | alternate squarespace entity |
| TUCOWS | google domains migration (common) |
| KEY_SYSTEMS | other migration source |

#### DeclaredSquarespaceDomainDnsRecordType

dns record types supported by squarespace.

| value | description |
|-------|-------------|
| A | ipv4 address |
| AAAA | ipv6 address |
| ALIAS | apex domain alias |
| CNAME | canonical name |
| MX | mail exchange |
| TXT | text record |
| SRV | service locator |
| NS | nameserver |
| CAA | certificate authority authorization |
| DS | delegation signer (dnssec) |
| DNSKEY | dns key (dnssec) |

#### DeclaredSquarespaceDomainLockReason

reasons a domain cannot be unlocked for transfer.

| value | description |
|-------|-------------|
| REGISTRATION_LOCK_60_DAY | newly registered within 60 days |
| TRANSFER_LOCK_60_DAY | transferred in within 60 days |
| CONTACT_UPDATE_LOCK | registrant info changed recently |

#### DeclaredSquarespaceTransferRequestStatus

transfer request status values.

| value | description |
|-------|-------------|
| REQUESTED | code requested, awaited via email |
| CODE_SENT | email with code was sent |
| IN_PROGRESS | transfer initiated at destination registrar |
| COMPLETED | transfer completed |
| CANCELLED | transfer cancelled within grace period |

## domain operations

### DeclaredSquarespaceDomainRegistration

| operation | semantics | description |
|-----------|-----------|-------------|
| getAll | retrieve | list all domains in account |
| getOne.byUnique | retrieve | get domain by name |
| set.upsert | mutate | update domain attributes (isLocked, dnssecEnabled, autoRenew) |

### DeclaredSquarespaceDomainDnsRecord

| operation | semantics | description |
|-----------|-----------|-------------|
| getAll | retrieve | list all dns records for a domain |
| getOne.byUnique | retrieve | get specific record by [domain, type, host] |

**note**: set/delete operations not implemented (out of scope for wish)

### DeclaredSquarespaceDomainTransferRequest

| operation | semantics | description |
|-----------|-----------|-------------|
| getAll | retrieve | list all transfer requests |
| getOne.byUnique | retrieve | get transfer request by domain |
| set.findsert | create-only | request transfer code (idempotent) |

**note**: findsert returns found request if already requested (no duplicate emails)

## relationships

### entity treestruct

```
DeclaredSquarespaceDomainRegistration (root)
├── unique key: { name }
├── mutable: isLocked, dnssecEnabled, autoRenew
│
├── DeclaredSquarespaceDomainDnsRecord[] (child, 0..n)
│   ├── unique key: { domain, type, host }
│   └── references parent via: domain: RefByUnique<DomainRegistration>
│
└── DeclaredSquarespaceDomainTransferRequest (child, 0..1)
    ├── unique key: { domain }
    └── references parent via: domain: RefByUnique<DomainRegistration>
```

### dependencies

1. **transfer request depends on domain unlock**
   - domain.isLocked must be false
   - otherwise: transfer code request fails

2. **transfer depends on dnssec disabled**
   - domain.dnssecEnabled should be false
   - otherwise: transfer may fail at destination registrar

3. **transfer request is one-per-domain**
   - only one active transfer request per domain
   - subsequent findsert returns found request

## composition for wish

the wish requests: enumerate and transfer domains out of squarespace.

### workflow composition

```
1. enumerate domains
   └─ daos.DeclaredSquarespaceDomainRegistration.get.all()

2. check dns configuration (for backup)
   └─ daos.DeclaredSquarespaceDomainDnsRecord.get.all({ domain: { name } })

3. prepare domain for transfer
   ├─ disable dnssec
   │  └─ daos.DeclaredSquarespaceDomainRegistration.set.upsert({ dnssecEnabled: false })
   └─ unlock domain
      └─ daos.DeclaredSquarespaceDomainRegistration.set.upsert({ isLocked: false })

4. request transfer code
   └─ daos.DeclaredSquarespaceDomainTransferRequest.set.findsert({ domain: { name } })
```

### declastruct pattern

```ts
// declare desired state
const resources = domains.flatMap((domain) => [
  new DeclaredSquarespaceDomainRegistration({
    ...domain,
    isLocked: false,
    dnssecEnabled: false,
  }),
  new DeclaredSquarespaceDomainTransferRequest({
    domain: refByUnique(domain),
  }),
]);

// apply
await declastruct.apply({ resources });
```
