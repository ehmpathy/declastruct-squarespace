### dependencies for domain transfer workflow

```
1. DomainRegistration.isLocked = false (unlock)
2. DomainRegistration.dnssecEnabled = false (disable DNSSEC)
3. TransferRequest.findsert() (request authorization code)
```

the transfer workflow requires sequential operations with dependencies:
- unlock before DNSSEC disable (recommended order)
- both unlock and DNSSEC disable before transfer code request
