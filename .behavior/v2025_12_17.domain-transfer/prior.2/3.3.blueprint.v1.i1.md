## contracts

### provider contract

```ts
const provider = getDeclastructSquarespaceProvider({
  account: { id: string, email: string },
  credentials: { email: string, password: string, totpSecret?: string },
  cache?: { directory?: string },
  browser?: { wsEndpointExtant?: string },
});

// returns
{
  context: ContextSquarespaceAgent,
  daos: {
    DeclaredSquarespaceDomainRegistration: Dao,
    DeclaredSquarespaceDomainDnsRecord: Dao,
    DeclaredSquarespaceDomainTransferRequest: Dao,
  },
}
```

### dao contract

```ts
interface DeclastructDao<T> {
  get: {
    one: {
      byUnique(input: { unique: RefByUnique<T> }, context): Promise<T | null>;
    };
    all(input: {}, context): Promise<T[]>;
  };
  set: {
    findsert?(input: { entity: T }, context): Promise<T>;
    upsert?(input: { entity: T }, context): Promise<T>;
    delete?(input: { unique: RefByUnique<T> }, context): Promise<void>;
  };
}
```

**key requirement: real domains must be fetched**

acceptance tests must verify real roundtrip behavior against a live squarespace account:

1. **live account required** — test account with at least one domain
2. **real resources returned** — `resources.acceptance.ts` must return actual domain declarations (not empty array)
3. **fail-fast on absent credentials** — no graceful skips; `UnexpectedCodePathError.throw` if credentials absent
4. **real scrape verified** — assert `domains.length > 0` to prove real data fetched

```ts
// resources.acceptance.ts must return real resources:
export const getResources = async () => {
  const [provider] = await getProviders();
  if (!provider) UnexpectedCodePathError.throw('credentials required for acceptance test');

  const domains = await provider.daos.DeclaredSquarespaceDomainRegistration.get.all({}, provider.context);
  if (domains.length === 0) UnexpectedCodePathError.throw('test account must have at least one domain');

  // return real resource declarations
  return domains.flatMap((domain) => [
    new DeclaredSquarespaceDomainRegistration({ ...domain, isLocked: false }),
    new DeclaredSquarespaceDomainTransferRequest({ domain: refByUnique(domain) }),
  ]);
};
```

**acceptance test must assert real data:**
```ts
then('returns non-empty domain list', () => {
  expect(prep.plan.changes.length).toBeGreaterThan(0);  // proves real domains fetched
});
```

## domain composition

```
DeclaredSquarespaceDomainRegistration (root entity)
├── unique key: { name }
├── mutable: isLocked, dnssecEnabled, autoRenew
├── readonly: status, lockReason, registrar, expirationDate, createdAt
│
├── DeclaredSquarespaceDomainDnsRecord[] (child entities, 0..n)
│   ├── unique key: { domain: { name }, type, host }
│   ├── mutable: value, ttl, priority
│   ├── readonly: isPreset
│   └── references parent via: domain: RefByUnique<DomainRegistration>
│
└── DeclaredSquarespaceDomainTransferRequest (child entity, 0..1)
    ├── unique key: { domain: { name } }
    ├── mutable: none (findsert only)
    ├── readonly: requestedAt, status
    └── references parent via: domain: RefByUnique<DomainRegistration>
```

## fix specifications

### fix 1: DeclaredSquarespaceDomainDnsRecord primary key

**file:** `src/domain.objects/DeclaredSquarespaceDomainDnsRecord.ts` line 51

**before:**
```ts
public static primary = ['domain'] as const;
```

**after:**
```ts
public static primary = ['domain', 'type', 'host'] as const;
```

**why:** primary key must uniquely identify one record; `['domain']` alone means all DNS records for a domain share the same primary key
