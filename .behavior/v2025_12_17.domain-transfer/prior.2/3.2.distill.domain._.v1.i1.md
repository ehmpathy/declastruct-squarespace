# distill: domain for declastruct-squarespace

## objective

distill domain objects, operations, and access patterns that fulfill the domain transfer wish.

---

## usecases mapped to contracts

### usecase: enumerate domains
```ts
const domains = await provider.daos.DeclaredSquarespaceDomainRegistration.get.all({}, context);
// returns: DeclaredSquarespaceDomainRegistration[]
```

### usecase: check dns records
```ts
const records = await provider.daos.DeclaredSquarespaceDomainDnsRecord.get.all(
  { domain: refByUnique(domain) },
  context
);
// returns: DeclaredSquarespaceDomainDnsRecord[]
```

### usecase: unlock domain
```ts
await provider.daos.DeclaredSquarespaceDomainRegistration.set.upsert(
  { entity: new DeclaredSquarespaceDomainRegistration({ ...domain, isLocked: false }) },
  context
);
```

### usecase: disable dnssec
```ts
await provider.daos.DeclaredSquarespaceDomainRegistration.set.upsert(
  { entity: new DeclaredSquarespaceDomainRegistration({ ...domain, dnssecEnabled: false }) },
  context
);
```

### usecase: request transfer code
```ts
await provider.daos.DeclaredSquarespaceDomainTransferRequest.set.findsert(
  { entity: new DeclaredSquarespaceDomainTransferRequest({ domain: refByUnique(domain) }) },
  context
);
```

## domain objects

### entities

#### DeclaredSquarespaceDomainRegistration

**identity:** uniquely identified by `name` (domain name)

**attributes:**
| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| name | string | no (unique key) | domain name (example.com) |
| status | RegistrationStatus | no (readonly) | ACTIVE, EXPIRED, PENDING |
| isLocked | boolean | yes | transfer lock enabled |
| lockReason | LockReason \| null | no (readonly) | why domain cannot be unlocked |
| registrar | RegistrarType | no (readonly) | backend registrar |
| expirationDate | string | no (readonly) | iso date of expiration |
| autoRenew | boolean | yes | auto-renewal status |
| dnssecEnabled | boolean | yes | dnssec enabled |
| createdAt | string \| null | no (readonly) | registration date |

**keys:**
```ts
public static primary = ['name'] as const;
public static unique = ['name'] as const;
public static readonly = ['createdAt', 'registrar', 'status', 'lockReason'] as const;
```

#### DeclaredSquarespaceDomainDnsRecord

**identity:** uniquely identified by `domain + type + host`

**attributes:**
| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| domain | RefByUnique<DomainRegistration> | no (key) | parent domain reference |
| type | DnsRecordType | no (key) | A, AAAA, CNAME, MX, etc |
| host | string | no (key) | subdomain or "@" for root |
| value | string | yes | record value |
| ttl | number | yes | time to live in seconds |
| priority | number \| null | yes | for MX, SRV records |
| isPreset | boolean | no (readonly) | squarespace preset record |

**keys:**
```ts
public static primary = ['domain', 'type', 'host'] as const;
public static unique = ['domain', 'type', 'host'] as const;
public static readonly = ['isPreset'] as const;
```

#### DeclaredSquarespaceDomainTransferRequest

**identity:** uniquely identified by `domain` reference

**attributes:**
| attribute | type | mutable | description |
|-----------|------|---------|-------------|
| domain | RefByUnique<DomainRegistration> | no (unique key) | domain to transfer |
| requestedAt | string | no (readonly) | iso date when code requested |
| status | TransferRequestStatus | no (readonly) | REQUESTED, CODE_SENT, etc |

**keys:**
```ts
public static primary = ['domain'] as const;
public static unique = ['domain'] as const;
public static readonly = ['requestedAt', 'status'] as const;
```

**note:** authorization code is sent via email, not returned directly

## domain operations

### DeclaredSquarespaceDomainRegistration operations

| operation | method signature | description |
|-----------|-----------------|-------------|
| getOne.byUnique | `({ unique: { name } }) => Promise<Domain \| null>` | lookup domain by name |
| getAll | `({}) => Promise<Domain[]>` | list all domains in account |
| set.upsert | `({ entity }) => Promise<Domain>` | update isLocked, dnssecEnabled, autoRenew |

**not supported:**
- set.findsert (domains created via registration, not this tool)
- set.delete (out of scope)

### DeclaredSquarespaceDomainDnsRecord operations

| operation | method signature | description |
|-----------|-----------------|-------------|
| getOne.byUnique | `({ unique: { domain, type, host } }) => Promise<Record \| null>` | lookup specific record |
| getAll | `({ domain }) => Promise<Record[]>` | list all dns records for domain |
| set.findsert | `({ entity }) => Promise<Record>` | find or insert record |
| set.upsert | `({ entity }) => Promise<Record>` | update record |
| set.delete | `({ unique }) => Promise<void>` | delete record |

### DeclaredSquarespaceDomainTransferRequest operations

| operation | method signature | description |
|-----------|-----------------|-------------|
| getOne.byUnique | `({ unique: { domain } }) => Promise<Request \| null>` | lookup transfer request |
| getAll | `({}) => Promise<Request[]>` | list all transfer requests |
| set.findsert | `({ entity }) => Promise<Request>` | request transfer code (idempotent) |

**not supported:**
- set.upsert (requests immutable once created)
- set.delete (not cancellable via this interface)

## relationships

### composition treestruct

```
DeclaredSquarespaceDomainRegistration (root entity)
├── DeclaredSquarespaceDomainDnsRecord[] (child entities, 0..n)
│   └── references parent via domain: RefByUnique
└── DeclaredSquarespaceDomainTransferRequest (child entity, 0..1)
    └── references parent via domain: RefByUnique
```

### reference pattern

all child entities reference parent domain via `RefByUnique<typeof DeclaredSquarespaceDomainRegistration>`:

```ts
// in child entity
domain: RefByUnique<typeof DeclaredSquarespaceDomainRegistration>
// resolves to { name: string }
```

### dependency order for transfer workflow

```
1. DomainRegistration.isLocked = false     (unlock)
2. DomainRegistration.dnssecEnabled = false (disable dnssec)
3. TransferRequest.findsert()               (request auth code)
```

unlock and dnssec disable must complete before transfer request.

## access layer (daos)

### dao structure

each dao exposes get/set operations with context passed per call:

```ts
export const DeclaredSquarespaceDomainRegistrationDao = {
  get: {
    one: {
      byUnique: async (input: { unique: RefByUnique<...> }, context) =>
        getOneDomain({ by: { unique: input.unique } }, context),
    },
    all: async (_input: {}, context) =>
      getAllDomains({}, context),
  },
  set: {
    upsert: async (input: { entity }, context) =>
      setDomain({ upsert: input.entity }, context),
    findsert: null,  // not supported
    delete: null,    // not supported
  },
};
```

### provider exposes daos

```ts
const provider = getDeclastructSquarespaceProvider({ ... });

// daos are accessible via provider
provider.daos.DeclaredSquarespaceDomainRegistration
provider.daos.DeclaredSquarespaceDomainDnsRecord
provider.daos.DeclaredSquarespaceDomainTransferRequest
```

## workflow composition

### batch transfer workflow

```ts
// 1. enumerate all domains
const domains = await provider.daos.DeclaredSquarespaceDomainRegistration.get.all({}, context);

// 2. declare desired state for each
const resources = domains.flatMap((domain) => [
  // unlock and disable dnssec
  new DeclaredSquarespaceDomainRegistration({
    ...domain,
    isLocked: false,
    dnssecEnabled: false,
  }),
  // request transfer code
  new DeclaredSquarespaceDomainTransferRequest({
    domain: refByUnique(domain),
  }),
]);

// 3. declastruct compares desired vs actual and applies changes
```

### single domain inspection

```ts
// lookup domain
const domain = await provider.daos.DeclaredSquarespaceDomainRegistration.get.one.byUnique(
  { unique: { name: 'example.com' } },
  context
);

// get dns records
const records = await provider.daos.DeclaredSquarespaceDomainDnsRecord.get.all(
  { domain: refByUnique(domain) },
  context
);

// check transfer status
const transfer = await provider.daos.DeclaredSquarespaceDomainTransferRequest.get.one.byUnique(
  { unique: { domain: refByUnique(domain) } },
  context
);
```
