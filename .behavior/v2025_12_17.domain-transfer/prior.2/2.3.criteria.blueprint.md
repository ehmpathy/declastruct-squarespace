# blueprint criteria: declastruct-squarespace domain transfer

## blackbox criteria satisfied

- usecase.1 = enumerate domains
- usecase.2 = check dns records
- usecase.3 = unlock domain for transfer
- usecase.4 = disable dnssec
- usecase.5 = request transfer authorization code
- usecase.6 = batch domain transfer workflow
- usecase.7 = authentication
- usecase.8 = bot detection resilience

all 8 usecases will be satisfied by the mechanisms declared below.

---

## subcomponent contracts

### provider contract

```
given('DeclastructSquarespaceProvider contract')
  then('exposes: getDeclastructSquarespaceProvider(config) => Provider')
  then('config requires: account.id, account.email, credentials.email, credentials.password, credentials.totpSecret')
  then('config optionally accepts: cache.directory, browser.wsEndpointExtant')
  then('provider.daos exposes: DeclaredSquarespaceDomainRegistration, DeclaredSquarespaceDomainDnsRecord, DeclaredSquarespaceDomainTransferRequest')
  then('context is baked into daos - no explicit context pass per call')
  then('throws AuthenticationError for invalid credentials')
  then('throws UnsupportedTwoFactorError for non-totp 2fa')
```

### domain registration dao contract

```
given('DeclaredSquarespaceDomainRegistration dao contract')
  then('exposes: get.all({}) => DeclaredSquarespaceDomainRegistration[]')
  then('exposes: get.byUnique({ name }) => DeclaredSquarespaceDomainRegistration | null')
  then('exposes: set.upsert({ entity }) => DeclaredSquarespaceDomainRegistration')
  then('entity includes: name, isLocked, dnssecEnabled, registrarType, lockReason')
  then('registrarType is one of: SQUARESPACE_DOMAINS_LLC, TUCOWS, KEY_SYSTEMS')
  then('throws DomainNotFoundError when domain does not exist')
  then('throws LockRestrictionError when unlock blocked by 60-day rule')
```

### dns record dao contract

```
given('DeclaredSquarespaceDomainDnsRecord dao contract')
  then('exposes: get.all({ domain: RefByUnique }) => DeclaredSquarespaceDomainDnsRecord[]')
  then('each record includes: type, name, value, ttl')
  then('type covers: A, AAAA, CNAME, MX, TXT, SRV, CAA, NS')
  then('throws DomainNotFoundError when domain does not exist')
```

### transfer request dao contract

```
given('DeclaredSquarespaceDomainTransferRequest dao contract')
  then('exposes: set.findsert({ entity: { domain: RefByUnique } }) => DeclaredSquarespaceDomainTransferRequest')
  then('exposes: get.byUnique({ domain: RefByUnique }) => DeclaredSquarespaceDomainTransferRequest | null')
  then('findsert returns extant request if already requested (idempotent)')
  then('throws DomainLockedError when domain is locked')
  then('throws DomainNotFoundError when domain does not exist')
```

### authentication layer contract

```
given('authentication layer contract')
  then('handles login flow with email + password')
  then('handles totp 2fa challenge via totpSecret')
  then('maintains session across operations')
  then('auto-refreshes session on expiry')
  then('throws UnsupportedTwoFactorError for sms or passkey 2fa')
```

### cache layer contract

```
given('cache layer contract')
  then('caches remote state for domains, dns records, transfer requests')
  then('cache has configurable TTL')
  then('cache can be cleared via config')
  then('cache directory is configurable')
```

### playwright scraper layer contract

```
given('playwright scraper layer contract')
  then('uses playwright-stealth to mask automation signals')
  then('operates sequentially with delays between actions')
  then('can connect to extant browser via wsEndpoint')
  then('throws BotDetectionError when automation is blocked')
```

## composition boundaries

### provider composes auth + cache + daos

```
given('provider composition')
  then('provider initializes auth layer on creation')
  then('provider initializes cache layer with config')
  then('provider exposes daos with auth and cache baked in')
  then('daos do not require explicit context on each call')
```

### daos compose scrapers + cache

```
given('dao composition')
  then('get operations check cache first')
  then('get operations scrape if cache miss')
  then('get operations update cache after scrape')
  then('set operations scrape current state first')
  then('set operations perform mutation via scraper')
  then('set operations invalidate relevant cache entries')
```

### scrapers compose playwright + auth

```
given('scraper composition')
  then('scrapers use shared authenticated browser context')
  then('scrapers use playwright-stealth via browser setup')
  then('scrapers handle navigation and element interaction')
  then('scrapers extract data into domain object shapes')
```
