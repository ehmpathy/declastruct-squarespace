# blackbox criteria: domain nameservers

## usecase.1 = get nameservers

```
given('a domain with squarespace default nameservers')
  when('nameservers are retrieved')
    then('nameservers is null')
      sothat('user knows domain uses squarespace DNS')

given('a domain with custom nameservers')
  when('nameservers are retrieved')
    then('nameservers is array of NS hostnames')
      sothat('user knows which external DNS provider is configured')
```

## usecase.2 = set custom nameservers (swap to cloudflare)

```
given('a domain with squarespace default nameservers')
  when('custom nameservers are set via upsert')
    then('nameservers is updated to custom array')
      sothat('domain now points to external DNS provider')
    then('operation is idempotent')
      sothat('repeat calls produce same result')

given('a domain with custom nameservers')
  when('different custom nameservers are set via upsert')
    then('nameservers is updated to new custom array')
      sothat('user can switch between external DNS providers')
```

## usecase.3 = reset to squarespace default (swap back)

```
given('a domain with custom nameservers')
  when('nameservers is set to null via upsert')
    then('nameservers is reset to squarespace default')
      sothat('domain returns to squarespace DNS management')
    then('operation is idempotent')
      sothat('repeat calls produce same result')

given('a domain with squarespace default nameservers')
  when('nameservers is set to null via upsert')
    then('nameservers remains null')
      sothat('no-op when already at default')
```

## usecase.4 = validation edgecases

```
given('a custom nameserver list with less than 2 entries')
  when('nameservers are set')
    then('operation fails with clear error')
      sothat('user knows minimum 2 nameservers required')

given('a custom nameserver with invalid FQDN format')
  when('nameservers are set')
    then('operation fails with clear error')
      sothat('user knows nameserver format is invalid')

given('an empty array for nameservers')
  when('nameservers are set')
    then('nameservers is treated as null (squarespace default)')
      sothat('empty array has clear semantics')
```

## usecase.5 = findsert semantics

```
given('a domain with no nameserver config')
  when('nameservers are set via findsert')
    then('nameserver config is created')
      sothat('new config can be established')

given('a domain with extant nameserver config')
  when('nameservers are set via findsert')
    then('extant config is returned unchanged')
      sothat('findsert does not overwrite')
```
