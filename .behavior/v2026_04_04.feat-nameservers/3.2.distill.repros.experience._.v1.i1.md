# distill: experience reproductions — nameservers

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| get nameservers | SDK | call `getNameservers({ by: { unique: { domain: { name } } } })` | returns entity with current nameservers or null | integration |
| swap to cloudflare | SDK | call `setNameservers({ upsert: { domain, nameservers: [...] } })` | returns entity with custom nameservers | integration |
| swap back to squarespace | SDK | call `setNameservers({ upsert: { domain, nameservers: null } })` | returns entity with null (squarespace default) | integration |
| findsert when absent | SDK | call `setNameservers({ findsert: {...} })` on domain with no config | config created | integration |
| findsert when present | SDK | call `setNameservers({ findsert: {...} })` on domain with config | extant config returned unchanged | integration |
| validation: < 2 nameservers | SDK | call `setNameservers` with 1 nameserver | throws validation error | unit |
| validation: invalid FQDN | SDK | call `setNameservers` with `invalid..nameserver` | throws validation error | unit |
| DAO integration | SDK | use `DeclaredSquarespaceDomainNameserversDao` | CRUD via declarative DAO | integration |

## journey test sketches

### journey 1: swap to cloudflare

```
given('[case1] domain with squarespace default nameservers')
  when('[t0] before any changes')
    then('getNameservers returns null or squarespace defaults')
  when('[t1] setNameservers upsert to cloudflare')
    then('returns entity with custom nameservers')
    then('nameservers array contains cloudflare NS')
  when('[t2] getNameservers after change')
    then('returns entity with custom nameservers')
```

#### step table

| step | action | user sees |
|------|--------|-----------|
| t0 | getNameservers | `{ domain: { name }, nameservers: null }` or squarespace defaults |
| t1 | setNameservers upsert | `{ domain: { name }, nameservers: ['ns1.cloudflare.com', 'ns2.cloudflare.com'] }` |
| t2 | getNameservers | `{ domain: { name }, nameservers: ['ns1.cloudflare.com', 'ns2.cloudflare.com'] }` |

#### input/output pairs

**t1 success case (snapshot target)**
```typescript
// input
const result = await setNameservers({
  upsert: {
    domain: { name: 'sunshineoceansurferturtles.com' },
    nameservers: ['ns1.cloudflare.com', 'ns2.cloudflare.com'],
  },
}, context);

// output
{
  domain: { name: 'sunshineoceansurferturtles.com' },
  nameservers: ['ns1.cloudflare.com', 'ns2.cloudflare.com'],
}
```

## critical paths

| critical path | description | why critical |
|---------------|-------------|--------------|
| swap to cloudflare | set custom nameservers on domain | primary usecase; if this fails, feature is useless |
| swap back to squarespace | reset nameservers to default | must be reversible; users need escape hatch |
| idempotent upsert | repeat upsert produces same result | automation requires idempotency; no accidental double-set |

## reproduction feasibility

### test utilities available

- `getSampleSquarespaceContext()` — provides context with credentials
- `requireSquarespaceCredentials()` — fail-fast if creds absent
- `useBeforeAll()` — shared test result capture
- `getNewLoggedInBrowserPage()` — browser session for scraper tests

### setup required

- squarespace account with test domain (`sunshineoceansurferturtles.com`)
- environment variables: `SQUARESPACE_EMAIL`, `SQUARESPACE_PASSWORD`, `SQUARESPACE_TOTP_SECRET`

### concrete test sketch

```typescript
describe('setNameservers', () => {
  requireSquarespaceCredentials();
  jest.setTimeout(600000);

  const TEST_DOMAIN = 'sunshineoceansurferturtles.com';
  const context = getSampleSquarespaceContext();

  given('[case1] domain with squarespace default nameservers', () => {
    when('[t0] setNameservers upsert to cloudflare', () => {
      const result = useBeforeAll(async () => {
        const ns = await setNameservers({
          upsert: {
            domain: { name: TEST_DOMAIN },
            nameservers: ['ns1.cloudflare.com', 'ns2.cloudflare.com'],
          },
        }, context);
        return { ns };
      });

      then('returns entity with custom nameservers', () => {
        expect(result.ns.nameservers).toEqual(['ns1.cloudflare.com', 'ns2.cloudflare.com']);
      });
    });
  });
});
```
